## Module Paths

You can access all needed folders using the following shortcuts

- Components: `@/comps/*`
- Layout: `@/layout/*`
- Documents: `@/docs/*`
- User Interace: `@/ui/*`
- Library: `@/lib/*`
- Pages: `@/pgs/*`
- API: `@/api/*`
- Styles: `@/css/*`
- Public Images: `@/img/*`
- Icons: `@/ico/*`
- Images: `@/imgs/*`
